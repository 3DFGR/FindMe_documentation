# Installation

FindMe is a Blender **extension** (add-on) and requires **Blender 5.0.0 or newer**.

## Requirements

| Requirement | Value |
| --- | --- |
| Blender version | 5.0.0 or newer |
| Supported platforms | macOS (Apple Silicon), Windows (x64) |
| Type | Add-on extension |

## Install from disk

1. Obtain the FindMe extension package (a `.zip` file).
2. In Blender, open **Edit ▸ Preferences ▸ Get Extensions** (or **Add-ons**).
3. Click the drop-down arrow in the top-right corner and choose **Install from Disk…**.
4. Select the FindMe `.zip` file.
5. Make sure the **FindMe** entry is enabled (checkbox ticked).

## Verify the installation

1. Open or hover over the **3D Viewport**.
2. Press `N` to open the sidebar.
3. You should see a **FindMe** tab. Click it to reveal the panel.

The panel header shows the installed version, for example `FindMe 1.4.0`.

```{note}
If the FindMe tab does not appear, confirm the extension is enabled in
**Preferences ▸ Add-ons** and that you are running a supported Blender version.
```

## Updating

Install the newer `.zip` the same way; Blender replaces the existing version.
Your enabled/disabled finder choices and per-finder settings are stored in the
add-on preferences and the scene, and are preserved across updates.
