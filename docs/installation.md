# Installation

FindMe is a Blender **extension** (add-on) and requires **Blender 5.0.0 or newer**.


## Install from disk

1. Obtain the FindMe extension package (a `.zip` file).
2. In Blender, open **Edit ▸ Preferences ▸ Get Extensions** (or **Add-ons**).
3. Click the drop-down arrow in the top-right corner and choose **Install from Disk…**.
4. Select the FindMe `.zip` file.
5. Make sure the **FindMe** entry is enabled (checkbox ticked).
