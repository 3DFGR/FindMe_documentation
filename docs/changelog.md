# Changelog

Notable changes per release. Versions follow the add-on's `blender_manifest.toml`.

## 1.4.0

- Scene-property integration for finders: per-finder settings shared across Object
  and Edit Mode via the ⚙ gear popup.
- New object filtering for the scan pipeline.
- UI refinements across the Object and Edit Mode panels.

## 1.1.4

- Trait filtering and match-count display in the panel.
- UI layout improvements.

## 1.1.2

- Renamed panel labels for clarity.
- Added the degenerate-vertex angle threshold.

## 1.0.6

- Added the *Scene Collection* inclusion option to collection-color filtering.
- Added the rat-nest connected-edge threshold.

## 1.0.1

- Collection color-tag filtering reworked.
- SPDX license headers added across the source.

## 0.9.x

- Consolidated Object and Edit Mode finders into dedicated modules with
  auto-discovery.
- Added the twisted-face threshold.
- Converted the add-on to the Blender extension format.
